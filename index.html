Tabla de Conceptos en HTML

Concepto Descripción Concepto Descripción
Tabla en HTML Estructura que organiza datos en filas y columnas. Formulario en HTML Estructura para recolectar datos del usuario.
Etiqueta table Define el inicio de una tabla. Etiqueta form Define el inicio de un formulario.
Atributos de table Propiedades como border, cellspacing, cellpadding. Etiqueta input Campo para ingresar datos en un formulario.
Etiqueta tr Define una fila dentro de la tabla. Etiqueta label Asocia un texto a un campo de formulario.
Atributo tr No tiene atributos específicos, pero puede usar atributos globales. Etiqueta select Desplegable para seleccionar opciones.
Etiqueta td y th td define una celda de datos, th define una celda de encabezado. Etiqueta textarea Área de texto para entradas más largas.
Atributos de td y th Atributos como colspan, rowspan, headers. Etiqueta fieldset, legend Agrupa elementos de un formulario.
Tablas avanzadas Uso de rowspan y colspan para combinar celdas. Etiqueta meter Representa un valor dentro de un rango conocido.
Etiqueta thead Define el encabezado de una tabla. Etiqueta progress Indica el progreso de una tarea.
Etiqueta tfoot Define el pie de una tabla. Etiqueta datalist Proporciona opciones predefinidas para un campo de entrada.
Etiqueta tbody Define el cuerpo de una tabla. Etiqueta summary Proporciona un resumen para un elemento detalles.
